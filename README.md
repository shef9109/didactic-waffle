# ⚔ Кузница Анкет

Генератор персонажных анкет для настольных ролевых игр.
Без регистрации, без серверов, без фреймворков — только HTML, CSS и JS.

## Возможности

- Динамические характеристики и снаряжение — добавляй и удаляй поля без ограничений
- 4 темы оформления: Средневековье, Магическая, Северная, Восточная
- Автосохранение в браузере через localStorage
- Экспорт в HTML и JSON, импорт JSON
- Печать / экспорт в PDF через браузер
- Несколько типов анкет: ролевая, гильдейская, боевая, магическая

## Структура проекта

├── index.html                  ← Главная страница
